Slide 02 // The Problem

The PII Crisis of Autonomous Agents

01

Relatable Pain

Booking clinic appointments or job forms requires sensitive user details (SSN, DOB, email) upfront.

02

Agent Eavesdropping

Standard agents pass raw PII to unsecure LLMs, leaving sensitive database traces in third-party environments.

03

Compliance Trap

Handling raw PII exposes platforms to massive HIPAA, GDPR, and security liability violations.

Slide 03 // The Solution

Privacy-Blind Egress Proxy

01

Local ECIES

User encrypts personal records locally prior to execution, passing only ECIES envelope to coordinator agent.

02

Placeholder Routing

The coordinator agent uses mock structures and placeholders like {{profile.email}}.

03

Edge Substitution

TEE enclave proxy intercepts request, decrypts profile in-memory, replaces placeholders, and dispatches to clinic.
